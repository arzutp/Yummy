# 🍽️ Yummy - Restoran Yönetim Sistemi (API & Web UI)

Yummy, modern web teknolojileri ve .NET 6 altyapısı kullanılarak geliştirilmiş kapsamlı bir restoran yönetim sistemidir. 
Bu proje, API eğitim süreçlerine ve HTTP standartlarına uygun olarak tasarlanmış olup, backend ve frontend entegrasyonunu uygulamalı olarak göstermeyi hedefler.

Proje; restoranların menülerini, rezervasyonlarını, müşteri geri bildirimlerini ve hizmetlerini yönetmelerini sağlayan güçlü bir **RESTful API** ile 
bu API'yi tüketen kullanıcı dostu bir **Web UI (MVC)** katmanından oluşmaktadır. Ayrıca yapay zeka (AI) entegrasyonu ile yenilikçi özellikler barındırır.

---

## 🚀 Öne Çıkan Özellikler

* **RESTful API Mimarisi:** HTTP isteklerine (GET, POST, PUT, DELETE) tam uyumlu servisler.
* **Web UI (MVC) Entegrasyonu:** API katmanını dinamik olarak tüketen yönetim arayüzü.
* **Yapay Zeka (AI) Entegrasyonu:** Lokal modellerle desteklenmiş yenilikçi özellikler.

---

## 🛠️ Kullanılan Teknolojiler
* **Backend:** .NET 6, Entity Framework Core
* **Frontend:** ASP.NET Core MVC
* **AI Servisleri:** OpenAI API, Hugging Face
---


## 🧱 Proje Yapısı
```
| Katman / Klasör | Türü | Açıklama |
| :--- | :--- | :--- |
| **Yummy.WebApi** | Backend | HTTP isteklerine yanıt veren RESTful API servis katmanı. |
| **Yummy.WebUI** | Frontend | API'yi tüketen kullanıcı dostu ASP.NET Core MVC arayüzü. |
```


## ✨ Projede Neler Yapıldı?
- Entity Framework Core ile veritabanı işlemleri gerçekleştirildi
- CRUD (Create, Read, Update, Delete) işlemleri uygulandı
- Frontend tarafında kullanıcı odaklı basit bir ui izlendi.
- OpenAI API ve Hugging Face servisleri projeye entegre edildi
